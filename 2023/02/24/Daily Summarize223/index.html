      • 统计语言模型

        • ```
          统计语言模型（Statistical Language Model，SLM）是一种基于概率统计的自然语言处理模型。它用来估计自然语言的序列的概率分布，即在给定先前的文本序列的情况下，下一个单词的出现概率是多少。
          通俗地说，统计语言模型是一种能够预测语言序列的模型。它的输入是一个文本序列，例如一个句子或一段话，输出是一个概率分布，表示在这个序列中下一个单词出现的可能性大小。

      • n元语言模型

        • ```
          n元语言模型是一种用来计算文本序列中的概率的模型，其中n表示模型中考虑的上下文信息的长度。以2元语言模型为例，它假设一个词的出现只与前面一个词有关，而与其他的词是独立的。因此，2元语言模型将文本序列中的每个词当做是在给定其前面一个词的情况下独立出现的。这样一来，我们就可以利用已知的2元组合（也就是相邻的两个词）的出现频率来计算一个文本序列的概率。

      • 词向量
      • 词向量是一种将文本中的单词映射为实数向量的技术。通过使用词向量，我们可以将单词之间的语义关系转化为向量空间中的距离或相似性，从而可以进行更加灵活和准确的文本处理和分析。

词向量的生成通常使用机器学习算法在大规模文本语料库上进行训练。这些算法根据单词在上下文中的出现情况和语义关系，生成每个单词的向量表示。这些向量通常由数百个到数千个维度组成，每个维度代表单词在不同语义维度上的重要性。

最常用的词向量算法是Word2Vec和GloVe。这些算法使用深度学习技术，如神经网络，来训练词向量，并且在自然语言处理领域得到了广泛应用。词向量被广泛用于各种文本任务，如文本分类、情感分析、机器翻译、问答系统等，以提高模型的性能和准确度。
